Party Drink Tracker - Public Assets
===================================

此資料夾包含 Web App 的 public 目錄結構。

重要注意事項：
由於執行環境限制，以下的 PNG 和 ICO 檔案目前僅為 1x1 像素的透明佔位檔案，
請務必使用繪圖軟體或線上轉檔工具，將 `app_icon.svg` 轉換為正確尺寸的圖檔並覆蓋它們。

檔案列表：
- public/
  - app_icon.svg    : SVG 原始向量檔 (請用此檔案產生 PNG)
  - manifest.json   : PWA 設定檔
  - index.html      : 網站入口 HTML 模板
  - logo192.png     : [請替換] 192x192 PNG 圖示
  - logo512.png     : [請替換] 512x512 PNG 圖示
  - favicon.ico     : [請替換] 網站 icon
